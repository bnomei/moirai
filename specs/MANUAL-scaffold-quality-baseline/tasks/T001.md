## Context

Create one package named `moirai`, edition 2021, with Rust 1.75 as the library MSRV. The feature
table is exactly `default = []`, `std = []`, and `testkit = []`; `alloc` is unconditional rather
than a Cargo feature. `src/lib.rs` must use `#![no_std]`, `extern crate alloc`, and
`#![forbid(unsafe_code)]`. Do not add runtime dependencies merely for convenience.

## DoD

- Cargo manifest and crate root compile as a no-default-features library.
- The feature contract is additive under `--all-features`.
- No forbidden sibling, framework, serialization, or proc-macro dependency is introduced.

## Validation

```sh
cargo check --lib --no-default-features
cargo check --all-features
cargo +1.75 check --lib --no-default-features
```

## Escalate If

Rust 1.75 cannot compile a required dependency or a required quality tool would force a runtime
dependency, feature coupling, or a higher library MSRV.
